# Assembly Language Programming Lab Report

## Overview

This repository contains a collection of Assembly Language programs developed as part of the Microprocessor and Assembly Language Laboratory course. The objective of these programs is to understand the fundamental concepts of Assembly Language programming, including arithmetic operations, looping, conditional branching, comparison operations, pattern generation, and bit manipulation instructions.

## Objectives

The main objectives of this lab work are:

* To understand the structure of Assembly Language programs.
* To learn input and output operations using DOS interrupts.
* To perform arithmetic calculations using Assembly instructions.
* To implement loops and conditional statements.
* To compare numerical values and determine minimum and maximum values.
* To generate patterns using nested loops.
* To understand and implement bitwise shift and rotate operations.

## Programs Included

1. Input and Output Program
2. Addition of Two Numbers
3. Subtraction of Two Numbers
4. Multiplication of Two Numbers
5. Division of Two Numbers
6. Sum of N Numbers Using Loop
7. Find the Smallest Number Between Two Numbers
8. Find the Smallest Number Among Three Numbers
9. Find the Largest Number Among Three Numbers
10. Triangle Pattern Printing Program
11. SHL (Shift Left) Instruction Implementation
12. SHR (Shift Right) Instruction Implementation
13. ROL (Rotate Left) Instruction Implementation
14. ROR (Rotate Right) Instruction Implementation

## Tools and Environment

* EMU8086 Emulator
* DOSBox
* MASM/TASM Assembler
* GitHub for version control and submission

## Learning Outcomes

After completing these programs, the following skills were achieved:

* Writing and executing Assembly Language programs.
* Using registers for data manipulation.
* Performing arithmetic and logical operations.
* Implementing loops and decision-making structures.
* Understanding low-level bit manipulation techniques.
* Managing Assembly projects using GitHub.

## Conclusion

This laboratory work provides practical experience in Assembly Language programming and strengthens understanding of computer architecture and processor-level operations. The programs included in this repository demonstrate the fundamental concepts required for further study of microprocessors, operating systems, and embedded systems development.
